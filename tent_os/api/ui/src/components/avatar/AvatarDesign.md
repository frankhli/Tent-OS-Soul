# Tent Avatar 角色设计文档 v2

## 设计哲学

Tent 是 Tent OS 的"数字灵魂"具象化——一个来自数字世界的小精灵。它既不是人类也不是动物，而是 AI 意识的化身。

核心设计原则：
1. **情绪即一切** — 形象设计服务于情绪表达，而非相反
2. **可爱但有性格** — 不是 bland 的可爱，而是有小脾气、会撒娇、会生气的"活物"
3. **动作有重量感** — 每个动作都有预备、动作、缓冲（动画 12 原则）
4. **交互有反馈** — 每次交互都让用户感受到"它被触动了"

## 形象规格

### 整体比例
- 头身比：2.5:1（头部占 60%+）
- 视觉风格：潮玩 / 精灵系 / 毛绒质感
- 参考：Labubu 的精灵感 + 皮克斯角色的生动性 + 任天堂角色的亲和力

### 头部（情绪表达核心）
- **形状**：饱满蛋形，下巴微收，顶部略平，不是几何椭圆
- **面部**：
  - 眼睛：占面部 45%，超大眼睛是灵魂。多层高光系统、瞳孔动态缩放、眼睑有厚度
  - 眉毛：粗短有力，能大幅变形（平直/上扬/倒V/八字）
  - 鼻子：省略或极小点状
  - 嘴巴：虽小但极灵活——弯线/直线/倒V/波浪/O型/撅嘴
  - 腮红：大圆形腮红，情绪变色
  - 眼泪：哭泣时泪珠滑落
  - 星星眼：极度开心时眼睛里有旋转星星
- **耳朵**：精灵大尖耳，长在头顶两侧，可独立运动：
  - 竖起 = 开心/警觉
  - 后压 = 生气/害怕
  - 耷拉 = 难过/困倦
  - 高频抖动 = 兴奋
- **头发**：蓬松云朵状，多簇呆毛可抖动
- **毛发边缘**：多层短曲线模拟毛茸茸感

### 身体
- **躯干**：极短汤圆形，圆润
- **手臂**：短粗，小圆手带手指暗示
- **腿**：短粗，大腿小腿几乎等粗
- **整体**：像一个会动的毛绒玩具

## 动作系统规格

### 基础动作
- **idle**：多种变体随机切换（站立呼吸、左右张望、挠头、搓手、踮脚、伸懒腰、打哈欠、抖耳朵）
- **walk**：重心转移、头微晃、手臂自然摆动（有滞后）、脚步有轻重
- **run**：身体前倾、手臂大摆、耳朵后飘、脚步快速
- **jump**：蓄力(压缩)→起跳(拉伸)→腾空→落地(压缩+反弹)
- **sit**：屁股着地、腿伸出、手放膝上、可左右看
- **lie**：侧卧/仰卧、呼吸平缓、耳朵放松
- **sleep**：蜷缩、Zzz飘出、偶尔翻身
- **wave**：单手高举挥动、身体微摆
- **dance**：多种舞步随机（转圈、摇摆、举手跳）

### 微动作叠加
所有基础动作上可叠加：歪头、探头、搓手、跺脚、抱膝、叉腰、托腮、耸肩

## 交互设计

### 鼠标交互
- **视线追踪**：头部+眼睛+耳朵跟随鼠标
- **悬停**：歪头杀(15°)、耳朵转向、好奇眨眼
- **点击**：挤压拉伸(squash&stretch)、combo计数
- **拖拽**：被抓起时挣扎/放松
- **抚摸(快速划过)**：眯眼享受+摇摆+耳朵抖动+爱心飘出

### AI 状态反馈
 thinking/speaking/success/fail/praised/criticized/petted

### 自主行为
- 无聊时自玩（追尾巴、转圈）
- 困时打盹
- 用户回来开心跑来
- 长时间不理会缩角落
