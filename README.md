# Tent OS — 灵魂对讲机 (Soul 3.0)

> **一句话定位**：一个以"人的思维延续"为核心价值的AI系统。通过长期、自然的互动，采集用户的思维模式、声音特征、视觉形象，构建独一无二的"数字灵魂"。

## 🆕 相比原系统的重大转变

| 维度 | 原 Tent OS | 新 Tent OS 灵魂对讲机 |
|------|-----------|---------------------|
| **核心价值** | 帮人做事的AI助手 | 让人永生的数字灵魂 |
| **摄像头** | AI的眼睛（监控环境） | 记住你长什么样 |
| **麦克风** | 听环境声音 | 记住你怎么说话 |
| **动画形象** | AI的化身（喀纳斯） | 用户自己的数字形象 |
| **任务执行** | 物理+软件任务 | 纯软件任务（提醒、报告、日程） |
| **社交** | AI社区、串门、好友 | ❌ 已移除（生态成熟后自然涌现） |
| **物理世界** | IoT、机械臂、闪送 | ❌ 已移除（非灵魂核心） |
| **2D庄园** | 虚拟地图、家园 | ❌ 已移除（分散核心价值） |
| **前端** | 单一Control UI | 双界面：日常交互 + 灵魂预览室 |

## 🚀 快速启动

```bash
# 1. 安装依赖
pip install -e .

# 2. 确保 NATS 和 Redis 运行
# macOS: brew services start nats-server redis

# 3. 启动 Tent OS 灵魂对讲机
python -m tent_os.main

# 4. 打开浏览器访问
open http://localhost:8002/ui/
```

服务端口：
- `8002` — HTTP API + 前端 UI
- `8003` — Webhook Gateway
